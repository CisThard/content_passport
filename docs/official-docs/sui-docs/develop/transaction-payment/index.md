# Paying for Transactions

A Sui transaction must pay for both the computational cost of execution and the long-term cost of storing the objects a transaction creates or mutates.

## [Gas FeesA Sui transaction must pay for both the computational cost of execution and the long-term cost of storing the objects a transaction creates or mutates.→](</develop/transaction-payment/gas-in-sui>)## [Object-Based Local Fee MarketsObject-based local fee markets limit the rate of transactions writing to a single shared object, preventing the network from becoming overloaded with checkpoints that take too long to execute.→](</develop/transaction-payment/local-fee-markets>)## [Gasless Stablecoin TransfersGasless stablecoin transfers on Sui enable peer-to-peer payments of qualified stablecoins to execute without paying gas fees in SUI.→](</develop/transaction-payment/gasless-stablecoin-transfers>)## [Sponsored TransactionsSponsored transactions let you pay gas fees on behalf of another user, reducing onboarding friction for users who don't own SUI.→](</develop/transaction-payment/sponsor-txn>)## [Gas SmashingSui optimizes coin management by combining multiple coins into a single object to pay for gas fees.→](</develop/transaction-payment/gas-smashing>)
