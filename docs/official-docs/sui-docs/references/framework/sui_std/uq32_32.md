# Module std::uq32_32

Defines an unsigned, fixed-point numeric type with a 32-bit integer part and a 32-bit fractional part. The notation [uq32_32](</references/framework/sui_std/uq32_32#std_uq32_32>) and [UQ32_32](</references/framework/sui_std/uq32_32#std_uq32_32_UQ32_32>) is based on [Q notation](<https://en.wikipedia.org/wiki/Q_\(number_format\)>). q indicates it a fixed-point number. The u prefix indicates it is unsigned. The 32_32 suffix indicates the number of bits, where the first number indicates the number of bits in the integer part, and the second the number of bits in the fractional part--in this case 32 bits for each.

## Struct UQ32_32​

A fixed-point numeric type with 32 integer bits and 32 fractional bits, represented by an underlying 64 bit value. This is a binary representation, so decimal values may not be exactly representable, but it provides more than 9 decimal digits of precision both before and after the decimal point (18 digits total).

## Function from_quotient​

Create a fixed-point value from a quotient specified by its numerator and denominator. [from_quotient](</references/framework/sui_std/uq32_32#std_uq32_32_from_quotient>) and [from_int](</references/framework/sui_std/uq32_32#std_uq32_32_from_int>) should be preferred over using [from_raw](</references/framework/sui_std/uq32_32#std_uq32_32_from_raw>). Unless the denominator is a power of two, fractions can not be represented accurately, so be careful about rounding errors.  
Aborts if the denominator is zero.  
Aborts if the input is non-zero but so small that it will be represented as zero, e.g. smaller than 2^{-32}.  
Aborts if the input is too large, e.g. larger than or equal to 2^32.
