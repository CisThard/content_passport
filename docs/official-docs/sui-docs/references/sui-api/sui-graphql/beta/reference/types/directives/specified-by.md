# specifiedBy

Exposes a URL that specifies the behavior of this scalar.
[code] 
    directive @specifiedBy(  
      url: String!  
    ) on SCALAR  
    
[/code]

### Arguments​

#### `specifiedBy.**url**` ● [`**String!**`](</references/sui-api/sui-graphql/beta/reference/types/scalars/string>) non-null scalar​

The URL that specifies the behavior of this scalar.
