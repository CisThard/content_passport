# ExecutionStatus

The execution status of this transaction: success or failure.
[code] 
    enum ExecutionStatus {  
      SUCCESS  
      FAILURE  
    }  
    
[/code]

### Values​

#### `ExecutionStatus.**SUCCESS**`​

The transaction was successfully executed.

#### `ExecutionStatus.**FAILURE**`​

The transaction could not be executed.

### Member Of​

[`TransactionEffects`](</references/sui-api/sui-graphql/beta/reference/types/objects/transaction-effects>) object
