# Kiosk

Learn how to use Kiosk, Sui's decentralized system for commerce applications, and how to extend its functionality with Kiosk apps.

## [Sui KioskKiosk is a decentralized system for commerce applications on Sui. Kiosk is a part of the Sui framework, native to the system, and available to everyone.→](</onchain-finance/kiosk/kiosk-example>)## [Kiosk AppsKiosk apps are a way to extend the functionality of Sui Kiosk while keeping the core functionality intact. You can develop apps to add new features to a kiosk without having to modify the core code or move the assets elsewhere.→](</onchain-finance/kiosk/kiosk-apps>)
